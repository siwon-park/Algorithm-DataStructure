# 08_Counter Clock Wise(CCW)

CCW 알고리즘은 3개의 점 A(x1, y1), B(x2, y2), C(x3, y3)가 있을 때, 이 점 3개를 이은 직선의 뱡향성을 알고자 할 때 사용하는 기하학 알고리즘이다.
